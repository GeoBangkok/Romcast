# RomCast

A SwiftUI application using SwiftData for data persistence.

## Features
- SwiftData integration
- NavigationSplitView interface
- Add/Delete items functionality

## Requirements
- iOS 17.0+
- Xcode 15.0+
- Swift 5.9+

## Installation
1. Clone the repository
2. Open RomCast.xcodeproj in Xcode
3. Build and run
